App Information

Match & Munch App

This is a simplified HTML preview of the Match & Munch React Native mobile app. The actual app would:

  • Run on iOS and Android devices
  • Use Firebase for authentication and data storage
  • Fetch movies from the TMDB API
  • Fetch food recipes from the Spoonacular API
  • Support real-time session sharing between users
  • Use React Native's gesture system for actual swiping

This preview demonstrates the basic user flow and interface design.
